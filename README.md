one-line-json
=============

It is used to turn a formated json to One line json

used to turn following

```json
{
    "id": 0,
    "name": "admin",
    "users": [
        {
            "id": 2,
            "name": "guest"
        },
        {
            "id": 3,
            "name": "root"
        }
    ]
}
```
to
```json
"{\"id\":0,\"name\":\"admin\",\"users\":[{\"id\":2,\"name\":\"guest\"},{\"id\":3,\"name\":\"root\"}]}"
```

`command(ctrl)+shift+p` then choose `oneline-JSON`
